# tecadi-test

Este projeto fora implementado para o teste da Tecadi

## Execute o docker

No diretório do projeto, escreva:

### `docker-compose up --build`

Rode o app neste endereço:
Abra [http://localhost:3000](http://localhost:3000) para ver em seu browser.

## Projeto

Neste projeto contém as operações de login/senha com alerta de acesso negado bem como o CRUD solicitado, e tambem alguns por menores para facilidades de uso do site em seu front end.
O projeto contém proteção de rotas e avisos para quaisquer operações, e paginação devida, filtro de saldo e numero de produtos por pagina.
